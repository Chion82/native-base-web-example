NativeBase for Web Example App
------------------------------
Component demo app using [react-native-web-extended](https://github.com/Chion82/react-native-web-extended) and [native-base-web](https://github.com/Chion82/native-base-web).

## Setup
```
npm i
```

## Develop
```
npm run dev
```

## Distribute
```
npm run build
```
